Main functions->

rov2_usingobjectdetection -> the position of the obstacle and the rover is taken from the 'test.mat' file generated by object detection algorithm.
rov2_vrep -> the position of the obstacle and the rover is taken from V-rep.

apf_LM -> provides potential field
apf_Path -> calculates path
apf_det -> gets the object handle and provides the detection parameter
apf_Move -> moves the rover
